# Solvedoku
A Sudoku solver written in C++ using the backtracking algorithm.

# Setup
Clone:\
`$ git clone https://github.com/JannikHv/solvedoku`

Switch to the repository's directory:\
`$ cd solvedoku`

Compile:\
`$ make`

Run:\
`$ make run`
